Tentang Saya

Saya adalah seorang pelajar jurusan RPL yang tertarik dengan pengembangan web dan desain grafis. Saya selalu berusaha belajar dan meningkatkan kemampuan saya di bidang ini.

Proyek

Berikut beberapa proyek unggulan yang pernah saya buat di jurusan RPL sebagai bentuk penerapan ilmu dan keterampilan dalam bidang pengembangan web dan desain grafis.

Gambar di atas adalah desain landing page website pemesanan tiket liburan. Desain ini menampilkan tampilan modern dengan fokus pada kemudahan pencarian destinasi, fitur booking, serta testimoni pengguna. Saya membuat desain ini menggunakan Figma

Gambar di atas adalah poster tata tertib Laboratorium RPL. Poster ini berisi aturan penting tentang ketentuan umum, keselamatan, keamanan data, dan sanksi pelanggaran. Saya membuat desain poster ini menggunakan CorelDraw.

Gambar di atas adalah proyek Python saya di Google Colab yang berisi tiga program: klasifikasi suhu, perhitungan luas dan keliling segitiga, serta penentuan grade nilai berdasarkan input pengguna.
